# Fixtures Métopes réalistes

Ce dossier contient deux fichiers réels issus d'un flux Métopes.

## Objectifs

- tester la lecture d'un vrai volume maître
- tester la présence de `xi:include`
- tester la détection d'une introduction autonome réelle
- éprouver le code sur un matériau plus riche que les fixtures synthétiques

## Remarque

Ces fichiers sont volontairement conservés proches de leur état réel. Ils servent surtout aux tests d'intégration et aux essais manuels.
